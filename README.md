## Cara Menjalankan

1. Clone repositori ini:
    ```bash
    git clone <url-repo>
    ```
2. Masuk ke direktori proyek:
    ```bash
    cd <nama-direktori>
    ```
3. Install dependency yang dibutuhkan
    ```bash
    pip install -m requirements.txt
    ```
4. Atur enviroment variable dengan membuat file `.env` di root folder
    ```env
    DB_NAME=
    DB_HOST=
    DB_PORT=
    DB_USER=
    DB_PASSWORD=
    SCHEMA=
    PRODUCTION=
    ```
